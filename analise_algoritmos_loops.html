Análise da Complexidade de Loops Aninhados

Caso 1: Loops Independentes

Complexidade: O(N²)

Ambos os loops executam N vezes independentemente.

Total = N × N = N²
for (i = 1; i <= N; i++) {
    for (j = 1; j <= N; j++) {
        // Instrução executada N² vezes
        contador++;
    }
}

Saída da Execução:

Explicação:

• Loop externo: N iterações

• Loop interno: N iterações para cada iteração do externo

• Total: N × N = N² operações

Caso 2: Loops com Limite N-1

Complexidade: O((N-1)²)

Ambos os loops executam N-1 vezes.

Total = (N-1) × (N-1) = (N-1)²
for (i = 0; i < N-1; i++) {
    for (j = 0; j < N-1; j++) {
        // Instrução executada (N-1)² vezes
        contador++;
    }
}

Saída da Execução:

Explicação:

• Loop externo: N-1 iterações (0 até N-2)

• Loop interno: N-1 iterações para cada iteração do externo

• Total: (N-1) × (N-1) = (N-1)² operações

Caso 3: Loop Interno Dependente

Complexidade: O(N²/2)

Loop interno diminui conforme i aumenta.

Total = (N² - N) / 2
for (i = 0; i < N-1; i++) {
    for (j = i+1; j < N; j++) {
        // Instrução executada (N²-N)/2 vezes
        contador++;
    }
}

Saída da Execução:

Explicação:

• Loop externo: N-1 iterações

• Loop interno: diminui de N-1 até 1 iteração

• Forma uma Progressão Aritmética: (N-1) + (N-2) + ... + 1

• Soma da P.A.: (N² - N) / 2
